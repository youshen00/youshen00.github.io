AJAX技术

AJAX介绍

AJAX即Asynchronous Javascript And XML(异步JavaScript 和 XML)。
他不是一种新的编程语言，而是通过异步实现的网页的局部更新技术。

核心对象

XMLHttpRequest对象是一个javascript对象，存在浏览器差异。简称xhr对象

AJAX编程思路


    //创建xhr对象
    var xhr;
        //屏蔽浏览器差异
        if(window.XMLHttpRequest){
            xhr = new XMLHttpRequest();
        }else{
            xhr = new ActiveXObject("Microsoft.XMLHTTP"); 
        }
    //发送请求参数
    xhr.open("GET|POST","url?参数");
    xhr.send(null);
    //处理响应并渲染页面
    xhr.onreadystatechange = function(){
        if(xhr.readyState == 4 && xhr.status == 200){
            console.log(xhr.responseText);
        }
    }

发送GET请求

    xhr.open("GET","url?参数=值");
    xhr.send();

发送post请求

    xhr.open("POST","url");
    // 设置发送请求的编码方式 默认方式
    xhr.setRequestHeader("content-type","application/x-www-form-urlencoded"); 
    xhr.send("参数 = 值")

AJAX的数据交换机制

JSON(javascript Object Notation js对象标记) 是一种轻量级的数据交换格式

1.

如果服务端响应的不再是字符串而是对象或者集合类型时，无法直接将对象响应给客户端

需要将对象类型转换成字符串类型

2.在后端代码使用json

导入alibaba  fastjson依赖 调用方法将对象转换成字符串

```
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>fastjson</artifactId>
    <version>1.2.47</version>
</dependency>

```

在Action方法中转换对象为字符串

```java
User user = new User();
String userJson = JSONObject.toJSONString(user);//fastjson
//设置response编码 防止中文乱码
response.setContentType("application/json;charset=UTF-8");
PrintWriter writer = response.getWriter();
writer.out(userJson);
```

3.在前端页面使用json

xhr.onreadystatechange = function(){
    if(xhr.readyState == 4 && xhr.status==200){
        var json = xhr.responseText;
        var userObj = eval("("+xhr.responseText+")");
        console.log(userObj.id);

    }
}

4.日期问题

```java
var userJson = JSONObject.toJsonStringWithDateFormat(user,"yyyy-MM-dd");
```

JQuery 对Ajax的封装

1.JQuery提供了全局函数

```java
$.ajax({
    type:"POST|GET",
    url:"",
    data:"name=张三|{key:value}",
    dataType:"JSON",
    success:function(result){
        console.log(result);
    }
})
```
